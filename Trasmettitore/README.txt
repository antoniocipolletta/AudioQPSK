Bisogna aggiungere al PATH di esecuzione di MATLAB la cartella functions che 
include le funzioni utilizzate.

TRASMETTITORE
	Utilizzare lo script tx_wired.m per generare il file .wav da riprodurre per trasmettere il file. 
	All'interno dello script è necessario cambiare il nome del file che si vuole trasmettere ed il 
	parametro nc (= numero di campioni per simbolo) che determina la velocità di trasmissione. Il numero
	nc deve essere necessariamente pari.
